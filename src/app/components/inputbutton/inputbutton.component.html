Radio Button

Os radio buttons são elementos de seleção que permitem aos usuários escolher apenas uma opção de um conjunto. Eles são representados visualmente como círculos preenchidos, onde apenas uma opção pode ser selecionada por vez.


HTML:

<form action="#">
 <div class="input-group">
  <input type="radio" id="option1" name="option" value="option1" checked>
  <label for="option1">Opção 1</label><br>
  <input type="radio" id="option2" name="option" value="option2">
  <label for="option2">Opção 2</label><br>
  <input type="radio" id="option3" name="option" value="option3">
  <label for="option3">Opção 3</label>
 </div>
</form>

Botão de Ligação

O botão de ligação é um elemento de interface que permite aos usuários alternar entre dois estados, como ligado/desligado, ativo/inativo, etc.

HTML:

<form action="#">
 <div class="input-group">
  <label for="toggle">Estado:</label><br>
  <input type="checkbox" id="toggle" name="toggle">
  <label for="toggle" class="toggle-label">Desligado</label>
 </div>
</form>
