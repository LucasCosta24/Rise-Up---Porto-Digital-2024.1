Button:

Em HTML, o Button pode ser definido pela tag de abertura <button> e a tag de fechamento </button> . Dentro de um elemento <button>, você pode colocar conteúdo, como texto ou imagens. Essa é a principal diferença entre esse elemento e os botões criados com o elemento <input>.
Botão Simples
HTML
<button class="basic">Simples</button>
CSS
.basic { background-color: #e0e0e0; }
Botão Fino
Link
HTML
<button class="thin">Fino</button>
<button class="thin basic disabled" disabled>Disabled</button>
<a class="thin basic" href="https://www.example.com/" target="_blank">Link</a>
CSS
.thin { padding: 5px 10px; border: 1px solid transparent; border-radius: 2px; font-size: 14px; line-height: 1.5; }
.thin.warn { background-color: #f44336; color: white; }
.thin.disabled { cursor: not-allowed; opacity: 0.5; }
