Elas são utilizadas em diversos contextos, desde a exibição de listas de produtos em lojas online até a apresentação de resultados de pesquisas em sites científicos.

Code:

O TypeScript aprimora a funcionalidade das tabelas HTML possibilitando a manipulação e interação dinâmica de dados. Você pode vincular dados de arrays ou objetos JavaScript às células da tabela, permitindo atualizações em tempo real e interações do usuário.

TS
export interface ColorNumbers {
nome: string;
posicao: number;
cor: string;
}
const COLOR_DATA: ColorNumbers[] = [
{posicao: 1, nome: 'Um', cor: 'Vermelho'},
{posicao: 2, nome: 'Dois', cor: 'Amarelo'},
{posicao: 3, nome: 'Três', cor: 'Azul'},
{posicao: 4, nome: 'Quatro', cor: 'Verde'},
{posicao: 5, nome: 'Cinco', cor: 'Roxo'},
{posicao: 6, nome: 'Seis', cor: 'Laranja'},
];
@Component({
selector: 'app-table',
styleUrl: 'table.component.css',
templateUrl: 'table.component.html',
standalone: true,
imports: [CdkTableModule],
})
expor class TableComponent {
displayedColumns: string[] = ['posicao', 'nome', 'cor'];
dataSource = new ExampleDataSource();
}

export class ExampleDataSource extends DataSource<ColorNumbers> {

data = new BehaviorSubject<ColorNumbers[]>(COLOR_DATA);

connect(): Observable<ColorNumbers[]> {
return this.data;
}

disconnect() {}
}

Tags Estruturais De Uma Tabela:

a estrutura básica de uma tabela consiste nas tags <tr>, <td> e as células de título <th>. Porém, além dessas, também há tags que servem para estruturar melhor a tabela. São elas:


<thead></thead> – Esta tag representa o cabeçalho da tabela, geralmente composta por células título.

<tbody></body>– Essa tag representa o corpo da tabela;

<tfoot></tfoot>– Essa tag representa o rodapé da tabela.

essas tags tem como principal efeito manter uma semântica adequada para suas tabelas.
