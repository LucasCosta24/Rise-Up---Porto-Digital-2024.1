Select:

A tag <select> em HTML cria um elemento de seleção, também conhecido como menu suspenso. Ele permite ao usuário escolher uma opção dentre uma lista predefinida. É uma ferramenta valiosa para coletar informações do usuário de forma controlada e eficiente.

Select Simples
HTML
<select class="select"></select>
<option value="">Selecione uma Opção</option>
<option value="Opção 1">Opção 1</option>
<option value="Opção 2">Opção 2</option>
<option value="Opção 3">Opção 3</option>

Select Agrupado

Agrupamento em HTML, no contexto da tag <select>, refere-se à técnica de organizar as opções de um menu suspenso em categorias distintas para facilitar a navegação e a seleção do usuário. Essa funcionalidade é particularmente útil quando se trabalha com um grande número de opções ou quando a categorização das opções agrega valor à experiência do usuário.

Select Agrupado
HTML
<select class="select" name="opcoes" id="opcoes">
<option value="">Selecione uma Opção</option>
<optgroup label="Grupo 1"></optgroup>
<option value="Opção 1">Opção 1</option>
<option value="Opção 2">Opção 2</option>
</optgroup>
<optgroup label="Grupo 2"></optgroup>
<option value="Opção 1">Opção 1</option>
<option value="Opção 2">Opção 2</option>
</optgroup>
</select>
