Input texto:

Em HTML, a tag <input type="text"> representa uma caixa de entrada de texto. Ela é utilizada para criar uma caixa de entrada de texto de tipo não específicado.

HTML:

<form action="#">
 <div class="input-group">
  <label for="username">Dados:</label>
  <input type="text" id="username" name="username" required>
 </div>
</form>

Input senha:

Em HTML, a tag <input type="password"> representa uma caixa de entrada de texto de tipo senha. Ela é utilizada para criar uma caixa de entrada de dados, onde o texto inserido não é visualizado pelo usuário.

HTML:

<form action="#">
 <div class="input-group">
  <label for="password">Senha:</label>
  <input type="password" id="password" name="password" required>
 </div>
</form>

Input data:

Em HTML, a tag <input type="date"> representa uma caixa de entrada de texto de tipo data. Ela é utilizada para criar uma caixa de entrada de dados, onde o texto é no formato dia/mês/ano.

HTML:

<form action="#">
 <div class="input-group">
  <label for="birthdate">Data de Nascimento:</label>
  <input type="date" id="birthdate" name="birthdate" required>
 </div>
</form>

Input Estilização:

Em HTML, nós podemos fazer a estilização de um input, com objetivo de deixar ela mais funcional para o usuário, estética e intuitiva.

CSS:

body {
 font-family: Arial, sans-serif;
 background-color: #f4f4f4;
 margin: 0;
 padding: 0;
}
h1 {
 color: blue;
}
p {
 color: green;
}
