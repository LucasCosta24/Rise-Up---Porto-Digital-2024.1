Chip:

Em HTML, o termo "chip" se refere a um elemento de interface gráfica de usuário (UI) que representa uma peça compacta de informação. Chips podem conter uma variedade de conteúdo, como texto, ícones, ou fotos, e geralmente são usados para representar entradas de usuário, tais como seleções, contatos, ou filtros.
