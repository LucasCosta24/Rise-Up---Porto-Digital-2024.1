Chip:

Em HTML, o termo "chip" se refere a um elemento de interface gráfica de usuário (UI) que representa uma peça compacta de informação. Chips podem conter uma variedade de conteúdo, como texto, ícones, ou fotos, e geralmente são usados para representar entradas de usuário, tais como seleções, contatos, ou filtros.

{{text1}}
{{text2}}

Se refere a um tipo de chip que exibe informações estáticas e não interativas. Esses chips podem ser úteis para mostrar informações que não precisam de interação do usuário, como tags ou rótulos.

HTML
<div class="chip" [ngStyle]="{{ '{' }}'background-color': bgColor{{ '}' }}">
 <div>{{text1}}</div>
</div>


CSS
.chip {
 display: inline-block;
 height: 32px;
 font-size: 14px;
 line-height: 32px;
 border-radius: 16px;
 background-color: #e0e0e0;
 margin: 5px;
}


TS
@Input() text1: string = "Chip um";
@Input('bg-color') bgColor: string = "#dce0dd";


Chip Modificável

Digite

Se refere a um tipo de chip que exibe informações Modificáveis.
Diferente do anterior, esse chip pode ser alterado pelo usuário
de forma dinâmica.

HTML
<div class="chip"(click)="handleClick()">
 <div "chip12" #chip>Digite</div>
</div>


CSS
.chip {
 display: inline-block;
 padding: 0 10px;
 height: 32px;
 font-size: 14px;
 line-height: 32px;
 border-radius: 16px;
 border: 1px solid #fb9127;
 background-color: #ee8e11;
 margin: 5px;
  margin-right: 10px;
 margin-bottom: 10px;
 font-family: Arial, sans-serif;
 cursor: pointer;
}

TS
@ViewChild('chip') chipRef: ElementRef|undefined;
@handleClick() {
  const newName = prompt('Digite um novo nome para o chip:');
  if (newName !== null && newName.trim() !== '') {
    const limiteCaracteres = 12;
    if(newName.trim().length > limiteCaracteres){
      alert(`Por favor, digite no máximo ${limiteCaracteres} caracteres.`);
      return;
    }
    if (newName.trim() !== '') {
      console.log(this.chipRef?.nativeElement);
      if (this.chipRef) {
        this.chipRef.nativeElement.innerHTML = newName;
      }
  }
  console.log(this.chipRef!.nativeElement);
  this.chipRef!.nativeElement.innerHTML = newName;
}
