Expanding Grid:

Um "expanding grid" (ou grade expansível) em HTML é uma interface de usuário onde você tem uma grade de itens e cada item pode ser expandido para mostrar mais detalhes ou conteúdo. Isso é frequentemente usado em aplicações da web para organizar e exibir informações de maneira mais compacta e interativa.
