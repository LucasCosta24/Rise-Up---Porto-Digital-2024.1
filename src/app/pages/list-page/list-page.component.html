List:

Em HTML, a tag <ul> representa uma lista não ordenada. Ela é utilizada para criar uma lista de itens onde a ordem não possui relevância específica. Cada item dentro da lista não ordenada é definido utilizando a tag <li> (item da lista).

Por padrão, os navegadores da web exibirão listas não ordenadas com marcadores (geralmente círculos, quadrados ou discos). É possível estilizar esses marcadores usando CSS se desejar uma aparência diferente.

Lista Simples
HTML
<ul role="list" class="list">
<li class="li">Item 1</li>
<li class="li">Item 2</li>
<li class="li">Item 3</li>
</ul>

Se for necessário inserir apenas uma linha individual de texto lista, o texto pode ser inserido diretamente no elemento <ul>.

Estilização de Listas

Listas podem ser ter diferentes estilos como ordenadas por números decimais, romanos, simbolos e etc. Exemplo de lista com estilo de números decimais:


CSS
.list {
list-style: decimal;
}

As listas podem ser apresentadas de diferentes formas a partir desse princípio, outros exemplos:


list-style: none; Estiliza a lista sem símbolos.

list-style: square; Estiliza a lista por símbolos em formato de quadrado.

list-style: circle; Estiliza a lista por símbolos em formato circular.

list-style: upper-roman; Estiliza a lista por símbolos em algarismo romano.

list-style: url; Estiliza a lista por símbolo externo inserido via url().

Lista Aninhada

Uma lista aninhada em HTML permite criar subníveis de hierarquia dentro de listas não ordenadas <ul> ou ordenadas <ol>. Isso é útil para organizar informações complexas de forma estruturada e facilitar a compreensão do usuário.


HTML
<ul role="list" class="list">
<li class="li">Item 1</li>
<li class="li">Item 2</li>
<ul>
<li class="li2">Item 2.1</li>
<li class="li2">Item 2.2</li>
</ul>
<li class="li">Item 3</li>
</ul>
CSS
.list {
list-style: none;
}
.li2 {
list-style: none;
}
Lista de Botões

Criar uma lista de botões em HTML envolve combinar as tags <ul> (lista não ordenada) ou <ol> (lista ordenada) com a tag <li> (item da lista) para definir cada botão. Dentro de cada <li>, você pode usar a tag <button> para criar o botão propriamente dito. Aqui está um exemplo básico:


HTML
<ul role="list" class="list">
<li class="li">
<button class="li-button-example">Botão 1</button>
</li>
<li class="li">
<button class="li-button-example">Botão 2</button>
</li>
<li class="li">
<button class="li-button-example">Botão 3</button>
</li>
</ul>
CSS
.li-button-example {
padding: 10px;
border: 1px solid black;
font-size: large;
background-color: white;
border-radius: 20px;
}
.li-button-example:hover {
cursor: pointer;
color: white;
background-color: rgb(0, 123, 255);
}
