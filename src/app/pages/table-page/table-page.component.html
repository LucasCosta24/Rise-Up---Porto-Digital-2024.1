Tags Estruturais De Uma Tabela :

a estrutura básica de uma tabela consiste nas tags <tr>, <td> e as células de título <th>. Porém, além dessas, também há tags que servem para estruturar melhor a tabela. São elas:


<thead></thead> – Esta tag representa o cabeçalho da tabela, geralmente composta por células título.

<tbody></body>– Essa tag representa o corpo da tabela;

<tfoot></tfoot>– Essa tag representa o rodapé da tabela.

essas tags tem como principal efeito manter uma semântica adequada para suas tabelas.
