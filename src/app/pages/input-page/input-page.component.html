Input texto:

Em HTML, a tag <input type="text"> representa uma caixa de entrada de texto. Ela é utilizada para criar uma caixa de entrada de texto de tipo não específicado.

HTML:

Input senha:

Em HTML, a tag <input type="password"> representa uma caixa de entrada de texto de tipo senha. Ela é utilizada para criar uma caixa de entrada de dados, onde o texto inserido não é visualizado pelo usuário.

HTML:

Input data:

Em HTML, a tag <input type="date"> representa uma caixa de entrada de texto de tipo data. Ela é utilizada para criar uma caixa de entrada de dados, onde o texto é no formato dia/mês/ano.

Input Estilização:
